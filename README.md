# UI Colorscheme Generator 

> A site for rapidly iterating UI colors schemes.   [Try it out](http://schemegen.surge.sh/)

![Example](/examples/demo.gif)

## Installation

> `npm install`

## Starting the project

`npm run start`

or if you're using yarn

`yarn start`
